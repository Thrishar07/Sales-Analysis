# Sales Analysis Project

A Python-based sales analysis project that processes sales data from CSV files, calculates item-wise and grand totals, and visualizes sales insights using bar graphs.

## Features

- Read sales data from CSV files
- Calculate total sales for each item
- Display formatted currency values
- Generate grand total of all sales
- Visualize sales data using Matplotlib bar charts
- Modular code using helper functions

## Technologies Used

- Python
- Pandas
- Matplotlib
